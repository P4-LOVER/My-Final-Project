# 🏠 Ames Housing: Capstone Data Analysis

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org)
[![Framework](https://img.shields.io/badge/Toolkit-Pandas%20%7C%20Seaborn%20%7C%20Scikit--Learn-orange)](https://pandas.pydata.org)

## 📋 Overview
This project focuses on the comprehensive analysis and processing of the **Ames Housing dataset**. Using advanced data science techniques, the workflow covers everything from raw data cleaning to deriving actionable insights through exploratory analysis.

---

## 📂 Project Structure

| File / Folder | Description |
| :--- | :--- |
| 🛠️ **01_cleaning.ipynb** | Data cleaning stage: handling missing values and outliers. |
| 🏗️ **02_features.ipynb** | Feature engineering: transformation, encoding, and selection. |
| 📊 **03_eda.ipynb** | Exploratory Data Analysis (EDA) and deep-dive visualizations. |
| 📁 **data/** | Contains the datasets (both **Raw** and **Processed**). |
| 📄 **report.pdf** | Final technical report with findings and conclusions. |

---

## 🛠️ Tech Stack
The project is built using the following tools and libraries:

* **Language:** Python
* **Analysis:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **Environment:** Jupyter Notebook
* **Version Control:** Git & GitHub
